## 📖 简介

一款为 [**AstrBot**](https://github.com/AstrBotDevs/AstrBot) 设计的群管理插件。群友可通过指令快速 @ 管理员并附带被引用消息；支持按举报次数阶梯式自动禁言。管理员可记录群友违规记录（警告/记过），累计达到阈值时自动踢出群聊。

---

## ✨ 功能特性

- 🚨 支持指令快速通知所有管理员
- 💬 支持回复某条消息进行举报
- 📝 显示举报人信息和举报原因
- 📶 支持按群等级限制使用举报指令
- 🛡️ 支持群聊白名单与指令黑白名单设置
- 👥 支持自定义通知对象和排除通知对象
- 📤 支持通知转发到目标群聊/私聊会话
- 🧷 支持引用举报时转发被引用原消息
- 🔇 支持时间窗口内按举报次数阶梯式自动禁言
- ⚠️ 支持群内警告记录、查看、撤销功能
- 👢 支持警告次数达到阈值后自动踢出群成员

## 🎮 指令模块

### 举报通知

直接发送举报指令：

```
/举报 广告
```

或使用别名：

```
/举办 广告
```

回复某条消息后使用举报指令，会引用被举报的消息：

```
[回复某条消息]
/举报 发广告
```

### 警告记录

管理员可通过以下指令记录和撤销群成员的警告信息，查看类指令允许所有群成员使用：

```
/警告 @某人 打广告
```

或使用别名：

```
/记过 @某人 打广告
```

如果是回复某条消息，则可以省略 `@目标`，会自动将被引用消息的发送者作为记录目标：

```
[回复某条消息]
/警告 打广告
```

查看单个群成员的记录：

```
/查看劣迹 @某人
```

或使用别名：

```
/查前科 @某人
```

查看当前群所有已有记录的群成员：

```
/查看劣迹群友
```

或使用别名：

```
/查看群友前科
```

撤销某个群成员最近一条记录：

```
/撤销警告 @某人
```

也可以追加序号，撤销展示结果中的第几条记录：

```
/撤销警告 @某人 2
```

## ⚙️ 配置说明

### 全局配置

| 配置项                          | 类型              | 默认值 | 描述                                       |
| :------------------------------ | :---------------- | :----- | :----------------------------------------- |
| **`whitelist_groups`**  | `list`          | `[]` | 仅在这些群组中启用。为空则全局启用。       |
| **`report_whitelist`**  | `list`          | `[]` | 举报白名单。列表中的用户无法被举报。       |
| **`command_blacklist`** | `list`          | `[]` | 指令黑名单。列表中的用户无法使用举报功能。 |
| **`group_rules`**       | `template_list` | `[]` | 按群配置举报规则。配置后优先使用规则匹配。 |

### 群规则配置

| 配置项 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| 生效群聊列表 | `list` | `[]` | 可填写多个群号，留空表示对所有群生效。 |
| 群等级限制 | `int` | `0` | 使用举报指令所需最低群等级，`0` 表示不限制（群主/管理员默认放行）。 |
| 通知对象设置 | `object` | `{}` | 通知对象（管理员/群主/仅自定义）、自定义对象、排除对象。 |
| 转发目标设置 | `object` | `{}` | 转发群聊/私聊会话、转发时是否在原群 @。 |
| 自动惩罚设置 | `object` | `{}` | 举报阶梯禁言（统计窗口、阈值、时长）、警告阈值踢出。 |

> [!TIP]
> **阶梯式禁言示例**：「自动惩罚设置」中设置**举报统计窗口**为 2 小时，**举报次数阈值**为 `3 5`，**禁言时长**为 `10 60`，表示同一用户 2 小时内被举报 3 次禁言 10 分钟，被举报 5 次及以上禁言 60 分钟。
